# Meter Dashboard User Manual

## Overview
- Manage Raspberry Pi devices and meter configurations via web admin.
- Deploy scripts/packages to Pi devices with one click.

## Getting Started
1. Log in to Django admin.
2. Add Raspberry Pi and Meter Device entries.
3. Use admin actions to activate/deactivate devices and meters.
4. Deploy OTA scripts using the OTADeployment section.

## Common Tasks
- Activate/deactivate devices and meters.
- Set up SSH keys for secure Pi access.
- Deploy scripts/packages to Pi devices.
- Monitor deployment status and timestamps.

## Troubleshooting
- If deployment fails, check SSH credentials and network connectivity.
- Refresh admin page to see updated status.
- For background tasks, ensure Celery and Redis are running.
