📊 All Available URLs for Testing

🌐 Web Interface:
• http://localhost:8000/device-config/ (this page)
• http://localhost:8000/device-config/pi/{id}/ (Pi details)
• http://localhost:8000/device-config/meters/ (Meter list)

📁 Export Utilities:
• http://localhost:8000/device-config/export/device-config/{pi_id}/ (JSON export)
• http://localhost:8000/device-config/export/system-config/{pi_id}/ (System config)

🔧 REST API Endpoints:
• http://localhost:8000/device-config/api/ (API root)
• http://localhost:8000/device-config/api/raspberry-pis/ (CRUD Pis)
• http://localhost:8000/device-config/api/raspberry-pis/{id}/test_connection/ (Test SSH)
• http://localhost:8000/device-config/api/raspberry-pis/{id}/setup_ssh_key/ (Setup SSH)
• http://localhost:8000/device-config/api/raspberry-pis/{id}/device_config_json/ (Device JSON)
• http://localhost:8000/device-config/api/raspberry-pis/{id}/system_config_json/ (System JSON)
• http://localhost:8000/device-config/api/meter-devices/ (CRUD meters)
• http://localhost:8000/device-config/api/meter-devices/by_pi/ (Meters by Pi)
• http://localhost:8000/device-config/api/meter-devices/available_models/ (Meter models)
• http://localhost:8000/device-config/api/system-configurations/ (CRUD system configs)
• http://localhost:8000/device-config/api/deployments/ (View deployments)

⚙️ Admin:
• http://localhost:8000/admin/ (Django admin interface)
